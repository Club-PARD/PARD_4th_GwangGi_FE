## Service Introduction

### 현재 나의 헌혈 가능 여부를 알려주고, 헌혈 챌린지(그룹)을 만들어 헌혈을 장려하는 웹/앱 서비스 , ‘블릿지’

> - 🧐 헌혈에 대한 잘못된 인식과 부정적인 경험 때문에 대구/ 경북 지역의 헌혈이 저조한 상황입니다.

> - 😔 헌혈을 하는 사람들을 접하여 이유와 동기를 깨닫고, 동기 부여와 인식 개선을 제공하고자 했습니다.

### 💡 이러한 문제를 해결하기 위해,

> 1. 서비스 자체에서 준비한 '헌혈 자가문진'을 진행하여, 헌혈 가능한 여부와 적격/부적격 사유에 대해서 직관적으로 제공합니다.

> 2. 함께 헌혈을 지속적으로 행하고 싶은 사람들을 모집하고 동참하는 챌린지를 통해서 헌혈을 격려합니다.

## Stack

### 🌟 Frontend(Web)

> Language : JavaScript

> Library & Framework : React, Styled-Components, Axios, dayjs, js-cookie, jwt-decode

> Deploy: Firebase

### 💻 Backend

> Language : Java

> Library & Framework : Spring Boot

> Database : MySQLDB

> ORM : JPA

> Deploy: Mysql(Storage)

> Infrastructure : AWS EC2, AWS Route 53, AWS VPC
